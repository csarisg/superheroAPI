# superheroAPI
pagina para elegir super heroes
En este proyecto vas a crear una aplicación que permita listar
superhéroes y obtener información sobre los mismos. Para eso
utilizarás una API REST gratuita con datos reales, realizarás
consultas a la misma pasándole distintos parámetros que permitan
personalizar la búsqueda, y mostrarás los datos obtenidos como
respuesta.
